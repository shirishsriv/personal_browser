# Personal Browser

## Overview
This project implements a multi-tab browser using PyQt6. The application allows users to open multiple tabs, navigate through various web pages, and manage their browsing experience effectively.

## Features
- Multi-tab support
- User-friendly interface
- Fast and responsive design
- Customizable settings

## Requirements
- Python 3.x
- PyQt6 library

## Installation
To install the required libraries, use the following command:
```
pip install PyQt6 PyQt6-WebEngine
```

## Usage
Run the application using:
```
python3 main.py
```
